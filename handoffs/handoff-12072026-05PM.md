## What Was Completed Since Last Handoff

### Order idempotency key implementation added

`POST /api/orders` now requires:

```text
Idempotency-Key
```

Implemented behavior:

- Missing header returns `400 Bad Request`.
- Header longer than 200 characters returns `400 Bad Request`.
- Same key with same request returns the existing order.
- Same key with different request returns `409 Conflict`.
- Successful order creation saves an idempotency record in the same transaction as the order and stock update.
